1. margin의 방향별 지정

  • margin-top: ;
  • margin-right: ;
  • margin-bottom: ;
  • margin-left: ;

2. margin의 값 갯수별 지정법

  • margin:n 사방;
  • margin:n n: 세로 가로;
  • margin:n n n;: 상 좌우 아래쪽
  • margin:n n n n;: 위 오른쪽 밑에 왼쪽

항상 수평 정중앙에 위치하는 블럭요소

margin: n auto;로 지정
(좌우 margin이 항상 동일하게)
