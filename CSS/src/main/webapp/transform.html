transform 요소의 변형

  • transform:translate(Xn,Yn); 위치 변형.
    (양수값은 우측이나 아래로, 음수값은 좌측이나 위로)
  • transform:scale(Xn,Yn); 크기 변형.
    (1을 기준으로 1보다 작으면 축소, 크면 확대)
  • transform:rotate(ndeg); 회전 변형. 각도 단위 deg 사용
    (양수값은 시계방향으로, 음수값은 반시계방향으로)
    x나 y를 추가하면 3차원 회전이 됨.
  • transform:skew(ndeg); 기울기 변형. 각도 단위 deg 사용
    (양수값은 시계방향으로, 음수값은 반시계방향으로)
    x나 y를 추가하면 3차원 회전이 됨.

    • transform-origin 변형의 기준점 변경

    기준점이 기본값은 정중앙임(50% 50%/)
    수평 수직 순서대로 방향값(left/center/right) 나 수치값(Xnpx/Xn%/Xnem Ynpx/Yn%/Ynem) 등으로 수정 가능

1. transform:translate(); 위치 변형

transform:translateX(-500px);

transform:translateY(500px);

transform:translateY(-500px);

transform:translate(1000px,-300px);

transform:translate(100%,100%);
