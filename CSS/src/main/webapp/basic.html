선택자 {속성:속성값;}

css 선택자 대표 3가지

  • tag 선택자 - 속성이 같은 모든 태그에 적용되므로 각각의 태그마다 다르게 설정 불가
  • id 선택자 - 태그에 id를 추가하고 id값을 선택자로 활용(#id값). - 중복 적용 불가(구조적 용도)
  • class 선택자 - 태그에 class를 추가하고 class 값을 선택자로 활용(.class값). - 중복 적용, 조합 가능(디자인 용도)

cSS 적용방식 3가지

  • 외부 CSS - 스타일 소스가 별개의 문서로 존재(*.css)하고 html문서의 head에 연결시켜 적용.
    (link href="*.css" rel="stylesheet")
    여러 페이지에 동시에 적용하고 수정 가능.
    유지보수에 유리.
    통일성 있는 디자인 가능.
  • 내부 CSS - html파일 head 내부에 style tag로 css소스를 적용
    해당 페이지에만 적용 됨
  • 인라인 CSS - 본문 html file 태그 뒤에 style="속성:속성값;" 방식으로 작성
    소스가 길어지고 유지보수에 불리, 통일성 불가
