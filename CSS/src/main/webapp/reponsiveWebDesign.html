Responsive Web Design

반응형 웹디자인 원칙 = mobile first

Viewport

뷰포트는 사용자가 웹 페이지에서 볼 수 있는 영역.
뷰포트는 기기마다 다르며, 컴퓨터 화면보다 모바일 화면에서는 더 작으므로
모바일 화면에서 웹 페이지를 축소 방지

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries 미디어 쿼리

@media특정 조건이 참인 경우에만 CSS 속성을 적용함

@media not|only mediatype and (media feature) { CSS-Code; }

Media Types

  • all - Used for all media type devices
  • screen - Used for computer screens, tablets, smart-phones etc.
  • print - Used for print preview mode

Media조건

  • width and height of the viewport - 뷰포트의 너비와 높이
  • orientation of the viewport (landscape or portrait) - 뷰포트 방향(가로 또는 세로)
  • resolution - 해상도
