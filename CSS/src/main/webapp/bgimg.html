background 요소의 배경

  • background-color:; 배경색상
  • background-image:url("이미지경로"); 배경이미지

background-image의 하위속성들

  • background-repreat:; 반복 지정
    • repeat (기본값) 양사방 반복
    • repeat-x 가로 반복
    • repeat-y 세로 반복
    • no-repeat 반복 없음
  • background-position:; 위치 지정
    • Xn Yn;
    • left/center/right top/center/bottom
  • background-size:; 크기 지정
    • Xn Yn; 가로 세로 크기를 직접 지정
    • contain; 그림 하나가 온전히 나옴(그림 우선)
    • cover; 이미지 하나로 영역 전체를 다 덮음
  • background-attachment:; 스크롤여부 지정
    • scroll - 영역 스크롤바에는 고정 / 브라우저 스크롤바에는 스크롤 됨
    • fixed - 영역 스크롤바와 브라우저 스크롤바에 모두 고정
    • local (특정영역에서)...
