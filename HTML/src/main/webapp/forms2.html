form2 (html5 input 요소들)

1. input type="time" - 시간 (hour:min am/pm)을 지정하는 요소

2. input type="date" - 날짜를 지정하는 요소

최소값은 min="nnnn-nn-nn"로 지정하고 최대값은 max="nnnn-nn-nn"로 지정

3. input type="datetime-local" - 날짜와 시간을 지정하는 요소

4. input type="week" - 주단위의 날짜를 지정하는 요소

5. input type="month" -

6. input type="number" - 숫자값을 지정하는 요소

최소값은 min=" "로 최대값은 max=" "로 지정, 증감 간격은 step=" "

7. input type="range" - 범위 내의 값을 지정하는 요소


8. input type="tel" - 연락처(전화번호)를 지정하는 요소


9. input type="email" - 연락처(이메일)를 지정하는 요소

이메일형식이 올바른지 확인
모바일브라우저에서 가상키패드나 @나 .com이 나타날 수 있음.

10. input type="url" - 웹주소를 지정하는 요소


11. input type="color" - 색상 값을 지정하는 요소




html 속성들

1. Autofocus (autofocus="autofocus")

2. Required (required="required")

3. 입력란의 설명 또는 예시

4. Autocomplete="on/off" - 자동완성기능을 켬/끔

5. Readonly - 읽기만 가능 (값이 전송 됨)

6. - 사용불가능한 요소로 지정 (값 전송 x)
