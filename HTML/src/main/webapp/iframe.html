웹페이지안의 독립된 공간

iframe 안에 불러올 수 있는 종류 (브라우저 처리 가능)

1. URL(웹주소)

2. *.htm/*.html (웹페이지) (웹)

3. *.git / *.jpg / *.png (image)

4. *.ogg / *.webm / *.mp4 (video)

5. *.txt (텍스트파일)

iframe안에 링크 열기

iframe에는 name의 값을 주고 그 값을 a의 target값과 동일하게 지정.
<iframe name="abc">
<a target="abc">
