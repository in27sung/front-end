링크를 걸어서 보여줄 수 있는 것

  • URL(Uniform Resource Locator)
  • web pages: *.htm, *.html
  • images: *.gif, *.jpg, *.png bird.png
  • videos: *.ogg, *.webm, *.mp4 ocean
  • text files: *.txt Bookmark

링크를 보여주는 방식 지정

  • 기본값. 현재 자리에 열기
  • 새 탭에서 열기
    • target="_blank"
  • target="_parent"(프레임구조)상위에서 열기
  • target="_top"(프레임구조)나뉨없이 전체에 열기

Web browser가 처리 불가능한 종류를 링크 걸면 다운로드 된다
