html문서를 구성하는 필수 태그들

<html>                       --------------- 웹문서(html)의 시작을 선언
    <head>                   --------------- 머리말의 시작을 선언
	<title> ... </title> --------------- 제목(브라우저의 탭영억, 북마크명)을 지정
    </head>                  --------------- 머리말의 시작을 선언
    <body>                   --------------- 본문의 시작을 선언
    </body>                  --------------- 본문의 끝을 선언
</html>                      --------------- 웹문서(html)의 끝을 선언

html = Hyper Text Marker Language의 약자.
웹페이지의 구조를 담당하는 언어.(The language for buildingg web pages>
명령어(Tag)로 구성되며
태그(각괄호)는 시작태그(Start tag)와 종료태그(End tag)의 한 쌍으로 구성됨.

저장은 확장자 *.htm이나 *.html로 지정하고
인코딩은 UTF-8로 지정, 메모장 = UTF-8(BOM)로 지정.
html소스는 공백이 스페이스바 하나 까지만 유지되고 엔터는 적용x

Type of Tag

문단태크 = <p>
줄바꿈태그 = <br>
수평선태그 = <hr>
공백유지태그 = <pre>
